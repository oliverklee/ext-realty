Compatibility with TYPO3 9LTS, 10LTS and 11LTS

For compatibility with TYPO3 CMS 9LTS, 10LTS and 11LTS, there is a crowdfunding campaign going on to fund the development work needed to make this extension compatible with these versions of TYPO3 CMS.

If you would like to see this extension work on TYPO3 9LTS, 10LTS and 11LTS, please consider contributing to the campaign and send the extension author an email: typo3-coding@oliverklee.de
